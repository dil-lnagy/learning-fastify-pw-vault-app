# Password manager

## Technologies used

- Next.js
- React.js
- Node.js
- Fastify
- TypeScript
- crypto.js

## Algorithms used

- SHA256 - Predictable password hash
- Argon2 - Unpredictable password hash for the database
- pbkdf2 - Generate the vault key
- AES256 - Encrypt and decrypt the vault
